# 🩺 Multi-Model Medical Image Diagnosis System

## [🚀 Live Demo]
https://burhan-medical-vit-resnet.streamlit.app

## 📌 Project Overview

This project is an end-to-end Medical AI system designed to assist in the detection of **Pneumonia** from Chest X-Ray images. It leverages a state-of-the-art **Ensemble Learning** approach, combining traditional convolutional features with global attention mechanisms to ensure high diagnostic reliability.

## 🏗️ Technical Architecture

To achieve robust performance, the system utilizes a dual-model ensemble:

  * **Vision Transformer (ViT-b-16):** Captures long-range dependencies and global context within the X-ray using self-attention.
  * **ResNet50:** Provides high-resolution spatial feature extraction through deep residual connections.
  * **Ensemble Inference:** Predictions are generated by aggregating softmax probabilities from both architectures, reducing individual model bias and achieving a higher confidence score.

### **Key Insights:**

  * **Performance:** The Vision Transformer (ViT) demonstrated superior performance in capturing subtle opacities compared to the standard ResNet baseline.

## 🔍 Explainable AI (XAI)

Medical AI must be interpretable. This system implements **Grad-CAM (Gradient-weighted Class Activation Mapping)** to provide visual evidence for its decisions.

  * **How it works:** The system generates a heatmap over the original X-ray, highlighting the specific regions (e.g., lung opacities) the model used to make its diagnosis.
  * **Benefit:** This allows clinicians to verify that the AI is focusing on biological markers rather than background noise (shoulders, clothing, etc.).

## 🛠️ Tech Stack
